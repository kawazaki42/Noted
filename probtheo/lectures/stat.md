__Вариационный ряд__ это ранжированный в порядке возрастания или убывания ряд
с соответствующими весами (частотами, частностями)

$x_i$ наблюдалось $n_i$ раз

$$n = \sum _{ k } ^{ i=1 } n_i$$

$n_i$ - частоты

относительная частота:

$$\omega = \frac{ n_i }{ n }$$

размах варьирования:

$$R = x_{max} - x_{min}$$

| ----- | ----- | ----- | ----- |

### среднее арифметическое вариационного ряда (или просто среднее)


$$\overline {X_В} = \frac{ \sum _{ i = 1 } ^{ n } n_i x_i  }{  n }$$

1) $\overline {X_В \pm C} = \overline {X_В} \pm C$

2) $\overline {CX} = C\overline {X}$

3) $(\overline {x_В \pm y_В}) = \overline {x_В} \pm \overline {y_В}$ 

4) Если X разбить на группы, то будет групповая и общая средняя

Медианой вариационного ряда называется значение признака,
приходящегося на середину ранжированного ряда наблюдений

$$Me^*, Med$$

Мода вариационного ряда - это значение варианта выборки,
у которого наибольшая частота

$$Mo ^*, Mod$$

Выборочная дисперсия

```math
\overline {D_В}
= \frac{ \sum _{ i = 1 } ^{ n } n_i x_i ^2 }{ n } - \overline{X_В}^2
```

$$\overline {\sigma_В} = \sqrt{ \overline {D_В} }$$

$$\overline {D_В} (CX) = C^2 \overline {D_В} (X)$$

$$\overline {D_В} (X + C) = \overline {D_В} (X)$$

Если x_i очень большие, то вводят условные варианты

$$U_i = \frac{ x_i - C }{ h }$$

где C, h - константы

$$\overline{U_В}, D_В (U), \sigma _В (U)$$

$$\overline{X_В} = h \cdot \overline{U_В} + C$$

$$D_В = h^2 D_В (U)$$

$$\sigma _В = h$$

Если x_i очень маленькие:

$$U_i = h \cdot x_i$$

$$\overline{X_В} = \frac{ 1 }{ h } \cdot \overline{U_В}$$

$$D_В = \frac{ 1 }{ h^2 } D_В (U)$$

## точечные и интревальные оценки для неизвестных параметров генеральского СОВКа

__Точечной__ называют статистическую оценку которая определяется одним числом.
При этом выборка должна быть достаточно большого объема.

__Несмещенной__ называют точечную статистическую оценку, математическое
ожидание которой равно оцениваемому параметру при любом объеме выборки.

__Смещенной__ называют статистическую оценку, математическое ожидание
которой не равно оцениваемому параметру.

В этом случае характеристики __выборочные__, или __эмпирические__.

Оценкой (несмещенной) мат. ожидания служит выборочное среднее

```math
\overline X_B = \frac{ 1 }{ n } (x_1 n_1 + ... + x_k n_k) = \sum _{ i = 1 } ^{ k } x_i n_i
```

- для дискретного ряда

```math
\overline X_B
= \frac{ 1 }{ n } (\tilde x_1 n_1 + ... + \tilde x_k n_k)
= \frac{ 1 }{ n } \sum _{ i = 1 } ^{ k } \tilde x_i n_i
```

- для интревального

$u_i$ или $\tilde x_i$ - середина интервала

```math
D_B
= \frac{ 1 }{ n } (x_1 ^2 n_k + ... + x_k ^2 n_k) - {\overline X_B}^2
= \frac{ 1 }{ n } \sum _{i = i} ^k x_i ^2 n_i - {\overline X_B}^2
```

- для дискретного

```math
D_B
= \frac{ 1 }{ n } ({\tilde x_1}^2 n_1 + ... + {\tilde x_k}^2 n_k) - {\overline X_B}^2
= \frac{ 1 }{ n } (\sum _{i = 1} ^k {\tilde x_i}^2 n_i) - {\overline X_B}^2
```

- для интревального

```math
D_{исп} = s^2 = \frac{ n }{ n-1 } D_B
```

$$\sigma_B = s = \sqrt s^2$$

### интревальные оценки

Используется, когда объем выборки не очень большой

Назвываются доверительными интервалами (синоним)

Определяется 2мя числами - концами интревала, которые
покрывают оцениваемый параметр

интревал доверительный если покрывается с определенной надежностью

__Интревальной оценкой__ с надежностью $\gamma$ математического ожидания $a$
нормально распределенного признака $X$ по выборочной средней $\overline X_B$
при известном среднеквадратичном отклонении $\sigma$ генерального СОВКа
будем называть интревал

```math
\begin{equation} \tag 1
\overline X_B - t(\frac{ \sigma  }{  \sqrt n })
< a
< \overline X_B + t(\frac{ \sigma  }{  \sqrt n })
\end{equation}
```

интрегалная фция лапласа

```math
\Phi _0 (t) = \frac{ \gamma }{ 2 }
```

Если среднеквадратичное отклонение не дано и объем выборки $n < 30$,
то доверительный интревал:

```math
\overline X_B - t_\gamma (\frac{ s  }{  \sqrt n })
< a
< \overline X_B + t_\gamma (\frac{ s  }{  \sqrt n })
```

Интревальной оценкой с надежностью $\gamma$,
среднеквадратичным отклонением $\sigma$
нормально распределенного количественного признака $X$
по исправленному выборочному среднеквадратичному отклонению $s=\sigma _{исп}$
служит доверительный интревал

```math
\begin{equation} \tag 2
\begin{cases}
s(1 - q) < \sigma < s(1 + q), & q < 1 \\
0 < \sigma < s(1 + q), & q > 1 \\
\end{cases}
\end{equation}
```

q - по таблице

$n$ и $\gamma$ - известны

#### Пример

найти доверительный интревал надежностью $\gamma = 0.95$
неизвестного мат. ожидания
нормално распределенного признака

$\sigma = 5$

$\overline X_B = 14$

$n = 25$

```math
% \begin{equation} \tag 1
14 - t(\frac{ 5  }{  \sqrt 25 })
< a
< 14 + t(\frac{ 5  }{  \sqrt 25 })
% \end{equation}
```

$$\Phi(t) = 0.95/2 = 0.475$$

$$t \approx 1.96$$

```math
% \begin{equation} \tag 1
14 - 1.96
< a
< 14 + 1.96
% \end{equation}
```

```math
12.04 < a < 15.96
```
