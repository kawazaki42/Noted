список домашек по ассемблеру
===

## 1. Арифметика


- возьмем 3 числа
- домножим каждое на свой порядковый номер (на 1, 2, 3) 
  - эти номера получаем при помощи `INC`
- все эти числа сложим
- эту сумму делим на 3
- из частного (результата деления) вычитаем остаток того же деления, умноженный на 3

## 2. Ветвления/условия

Проверить четность числа.

Дополнительно: проверить кратность числа другому числу.

## 3. Циклы

Прибавить к некоторому числу двойку 10 раз.

## 4. Функции

Создать функцию (на ассемблере!) которая складывает два числа (свои аргументы).

Аргуметны передавать через стек (`PUSH`/`POP`).

С её помощью прибавить к некоторому числу другое число 10 раз (используя цикл).
