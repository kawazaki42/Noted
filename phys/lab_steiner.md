Лабораторная работа 9
===

Определение момента инерций и проверка теоремы Штейнер методом крутильних
колебаний
===

__Цель работы__: экспериментальное определение моментов инерции для платформы
с различным расположением грузов на ней и проверка на основе этих данных
теоремы Штейнера

__Принадлежности__: трифилярный подвес, секундомер, два груза, линейка

| Поступательное движение                               | Вращательное движение                                       |
| ----------------------------------------------------- | ----------------------------------------------------------- |
| Координата $x$                                        | Угол поворота $\varphi$                                     |
| Линейная скорость $\overline V$                       | Угловая скорость $\omega$                                   |
| Линейное ускорение $\overline a = d \overline V / dt$ | Угловое ускорение $\varepsilon = d \omega / dt$             |
| Сила $\overline F$                                    | Момент силы $\overline M = [\overline r \cdot \overline F]$ |
| Масса $m$                                             | Момент инерции $J$                                          |
