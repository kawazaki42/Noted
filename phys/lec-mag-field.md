# мангнитное поле постоянного электрического поля

В 1820 г. Эрстед открыл явление возникновения магнитного поля
вокруг проводника, по которому идет электрический ток


Силовые линии магнитного поля всегда замкнуты сами на себе
- Магнитное поле носит исключительно вихревой характер,
- потому что в природе не существует _монополей_ (пока не открыты)

![alt text](<images/силовые линии маг поля.excalidraw.svg>)

Силовые линии магнитного поля проводятся таким образом
- чтобы вектор магнитной индукции (B\[Тл], силовая характеристика маг. поля)
- в каждой точке пространства был направлен по касательной к этим линиям

Силовые линии вокруг электрического тока проводятся по правилу правого винта
(правого буравчика)

Дальнейшие эксперименты позволили установить т.н. закон Био-Савара-Лапласа,
- который позволяет рассчитать величину магнитной индукции,
  - создаваемой проводником с током

$$\overline B = \int _{ l } ^{  } d \overline B$$

```math
d \overline B
= \frac{ \mu _0 \mu  }{  4 \pi } \cdot \frac{ I dl \cdot \sin \alpha  }{  r^2 }
```

\alpha - угол между направлением $dl$ и радиус-вектором

если альфа стремится к нулю, то и синус тоже стремится к нулю

следовательно, удаленные от точки наблюдения участки тока,
для которых $r \rarr \infty$, $\alpha \rarr 0$,
практически не будут давать вклада в магнитное поле
в точке наблюдения

отсюда следует, что основной вклад в создание магнитного поля
дают участки, для которых $\alpha \rarr 90 \degree$

Idl - элемент тока

$\mu _0 = 4 \pi \cdot 10^{-7}$ Гн/м

\mu - магнитная проницаемость среды

для ваккуума:
\mu = 1

все остальные вещества по характеру отклика на магнитное поле делятся на
- ферромагнетики
  - намагничиваются
- парамагнетики
- диамагнетики

магнитная проницаемость характеризует способность вещества намагничиваться

```math
E = \frac{ 1 }{  4 \pi \varepsilon _0 \varepsilon }
\cdot \frac{ q_1 \cdot q_2  }{  r^2 }
```

диэлектрическая проницаемость \varepsilon vs магнитная

> Используя закон Био-Салара-Лапласа можно легко рассчитать
> индукцию магнитного поля в центре кругового тока

\alpha = 90 \degree = const

r = R = const

I = const

$$B = \int _{ l } ^{  } \frac{ \mu _0 \mu  }{  4 \pi } \cdot \frac{ I dl }{ R^2 }$$

$$B = \frac{ \mu _0 \mu  }{  4 \pi } \cdot \frac{ I }{ R^2 } \int _{ l } ^{  } dl$$

$$L=2\pi R = \int _{ l } ^{  } dl$$

$$\boxed{B =  \frac{ \mu _0 \mu I }{ 2 R }}$$

Рассчитать магнитную индукцию B кольцевого тока

I = 100 мА = 0.1 А

R = 10 см = 0.1 м

$$B = \frac{ 4 \pi \cdot 10^{-7} Гн/м * 1 * 0.1 А }{ 2 * 0.1 м }$$

$$B = \frac{ 4 \pi \cdot 10^{-7} Гн/м * 1 А }{ 2 м }$$

$$B = \frac{ 2 \pi \cdot 10^{-7} Гн * 1 А }{ м^2 }$$

такого магнитного поля достаточно чтобы конкурировать с магнитным полем
Земли

## Сила Ампера

в 1824 г. Анри Ампер установил что на проводник с током,
находящийся в магнитном поле
действует сила

$$F_A = Idl B \cdot \sin \alpha$$

$$\alpha = \overgroup{dl, B}$$

$$F_A = Idl \cdot B$$

$$B = \frac{ F_A  }{  Idl } \[Тл]$$

$$E = \frac{ F_к  }{  q }$$

Сила ампера направлена по правилу левой руки
- 4 пальца левой руки направить по силе тока (движение положительных частиц)
- линии магнитной индукции входят в ладонь
- тогда большой палец левой руки отогнутый на 90 градусов
- покажет направление силы ампера
- то же для силы лоренца

Сила лоренца - действует на электрический заряд
(положительный или отрицательный),
двигающийся в магнитном поле

$$I = q/t$$

$$F_л = q/t * dl * B * \sin \alpha$$

$$u = dl/t$$

$$F_л = qu * B * \sin \alpha$$

$$\angle \alpha = \overgroup{u, b}$$
