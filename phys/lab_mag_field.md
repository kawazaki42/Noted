# Лабораторная работа 209. Определение горизонтальной составляющей поля Земли

Цель: практическое освоение метода измерения горизонтальной составляющей
индукции магнитного поля Земли с помощью колец Гельмгольца и компаса.

Приборы и оборудование: блок питания, сменный модуль, магазин сопротивления,
измерительный прибор.

$\vec H_0$ - горизонтальная составляющая вектора напряженности магнитного поля
Земли

$\vec H_1$ - искуственное магнитное поле (перпендикулярно к $H_0$)
- создается с помощью колец Гельмгольца
  - которые представляют собой две одинаковые катушки,
  - расположенные симметрично оси магнитной стрелки компаса

$\varphi$ - угол стрелки компаса

$$\ctg \varphi = H_0/H_1$$

$$H_1 = 2 H_k = 2nH_B$$

- $H_k$ - напряженность магнитного поля катушки
- $n$ - число витков в катушке
- $H_B$ - напряженность магнитного поля от одного витка

<!-- $\vec H_B$ вычисляется по принципу суперпозиции и закона Био-Савара-Лапласа -->

По принципу суперпозиции: $\vec H_B$ на оси витка в точке A равна
геометрической сумме напряженностей полей $d \vec H$, создаваемых
элементами витка $dl$ в этой точке

$$H_B = \int d \vec H_B$$

По закону Био-Савара-Лапласа:

```math
d \vec H
= \frac{ 1 }{ 4\pi } \frac{ [\overrightarrow{I \, dl_B}, \vec r] }{ r^3 }
```

Векторы $d \vec H$ образуют коническую поверхность

Запишем уравнение в проекциях:

$$H_x = \int dH_x$$

$$H_y = \int dH_y$$

$$H_z = \int dH_z$$

По теореме Пифагора:

$$H_B = \sqrt{ H_x ^2 + H_y ^2 + H_z ^2 }$$

Для любых двух диаметрально противоположных элементов витка $dl_1$, $dl_2$:
<!-- проекции векторов $d \vec H_1$ и $d \vec H_2$ -->

$$| d  H_{1y} | = | d H_{2y} |$$

$$d H_{1y} + d H_{2y} = 0$$

$$H_y = \int dH_y = 0$$

то же идля оси z

$$H_z = \int dH_z = 0$$

Проекции на ось x имеют одинаковые знаки

$$H_B = H_x = \int d H_x$$

По закону Био-Савара-Лапласа:

$$dH = \frac{ I \, dl \, \sin \alpha  }{  4 \pi r^2 }$$

проекция:

$$dH_x = \frac{ I \, dl \, \sin \alpha \cos \beta }{  4 \pi r^2 }$$

- $\alpha = 90 \degree$ - угол между $d \vec l$ и $d \vec r$
- $\sin \alpha = 1$
- $\beta$ - угол между $d \vec H$ и осью x
  - одинаков для всех элементов тока

подставляем

```math
H_B = \int \frac{ I \, dl \, \cos \beta }{  4 \pi r^2 }
= \frac{ I \cos \beta  }{  4 \pi r^2 } \int _{ 0 } ^{ 2 \pi R } dl
= \frac{ IR \cos \beta  }{  2 r^2 }
```

- R - радиус витка

В треугольнике OBA:
- O - центр основания цилиндра
- B - точка на окружности основания цилиндра
- A - точка измерения магнитного поля
- угол $B = \beta$
- $\cos \beta = R/r$
- $r = \sqrt{R^2 + h^2} = (R^2 + h^2)^{1/2}$

$$H_B = \frac{ IR^2  }{  2 r^3 }$$

$$H_B = \frac{ IR^2  }{  2 (R^2 + h^2)^{3/2} }$$

компас находится в центре витка с током

$$h = 0$$

$$B = \mu _0 \mu I / 2R$$

$$B = \mu _0 \mu H$$

$$H = I/2R$$

магнитные поля усиливают друг друга

$$H_1 = \frac{ InR^2  }{ (R^2 + h^2)^{3/2} }$$

$$H_1 = In/R$$

$$H_0 = \frac{ InR^2  }{ (R^2 + h^2)^{3/2} } \ctg \varphi$$

$$H_0 = \frac{ In }{ R } \ctg \varphi$$
