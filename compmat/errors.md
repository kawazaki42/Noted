Задание 5. Оценить абсолютную погрешность функции
---

$$x_1 = 0.75 \pm 0.01$$

$$x_2 = 1.723 \pm 0.002$$

$$x_3 = 7.8 \pm 0.4$$

$$f(x_1, x_2, x_3) = \frac{x_1 + x_2 + x_3}{x_1x_2x_3}$$

<!--
(0.75 + 1.723 + 7.8) / (0.75 1.723 7.8)
10.273 / 10.07955
1.0191923251
-->

$$f(0.75, 1.723, 7.8) \approx 1.0191923251$$

Теорема о предельной абсолютной погрешности функции:

```math
\Delta _f = \sum _{ k=1 } ^{ n }
\left| \frac{ \partial f }{ \partial x _k } \right| \cdot \Delta_{x_k}
```

Ищем производные:

```math
\frac{ \partial }{ \partial x_1 } \frac{x_1 + x_2 + x_3}{x_1x_2x_3} =
\frac{ \partial }{ \partial x_1 } \frac{x_1 + C}{x_1 \cdot D} =
```
```math
=
\frac{ 1 }{ x_2x_3 }\frac{ \partial }{ \partial x_1 } \frac{x_1 + C}{x_1} =
\frac{ 1 }{ x_2x_3 }\frac{ \partial }{ \partial x_1 } (1 + \frac{C}{x_1}) =
```
```math
=
\frac{ 1 }{ x_2x_3 }\frac{ \partial }{ \partial x_1 } (1 + {C}{x _1 ^{-1}}) =
\frac{ C }{ x_2x_3 }\frac{ \partial }{ \partial x_1 } {x _1 ^{-1}} =
```
```math
=
\frac{ x_2 + x_3 }{ x_2x_3 } \cdot (-1x _1 ^{-2}) =
-\frac{ x_2 + x_3 }{ x_2x_3 } \cdot x _1 ^{-2} =
```
```math
=
-\frac{ x_2 + x_3 }{ x _1 ^{2} x_2x_3 }
```

Поскольку умножение и сложение коммутативны,
частные производные будут "симметричны":

```math
\frac{ \partial f(x_1, x_2, x_3) }{ \partial x_1 } =
-\frac{ x_2 + x_3 }{ x _1 ^{2} x_2x_3 }
```

```math
\frac{ \partial f(x_1, x_2, x_3) }{ \partial x_2 } =
-\frac{ x_1 + x_3 }{ x _1  x_2 ^{2} x_3 }
```

```math
\frac{ \partial f(x_1, x_2, x_3) }{ \partial x_3 } =
-\frac{ x_1 + x_2 }{ x _1  x_2 x_3 ^{2} }
```

Ищем значения производных в точке:

<!--

-(0.75 + 7.8)/(0.75 1.723^2 7.8)
-(0.75 + 1.723)/(0.75 1.723 7.8^2)

-(0.75 + 1.723)/(0.75 1.723 7.8^2) 0.4

```math
\frac{ \partial f(0.75, 1.723, 7.8) }{ \partial x_1 } =
-\frac{ 1.723 + 7.8 }{ 0.75 ^{2} \cdot 1.723 \cdot 7.8 }
\approx -1.2597123218
```

```math
\frac{ \partial f(0.75, 1.723, 7.8) }{ \partial x_2 } =
-\frac{ 0.75 + 7.8 }{ 0.75 \cdot 1.723 ^{2} \cdot 7.8 }
\approx -0.4923111748
```

```math
\frac{ \partial f(0.75, 1.723, 7.8) }{ \partial x_3 } =
-\frac{ 0.75 + 1.723 }{ 0.75 \cdot 1.723 \cdot 7.8 ^{2} }
\approx -0.0314549044
```

Подставляем:

```math
\Delta _f =
\left| -1.2597123218 \right| \cdot 0.01 +
\left| -0.4923111748 \right| \cdot 0.002 +
\left| -0.0314549044 \right| \cdot 0.4
\approx 0.0261637073
```

> Ответ:
> $$f(0.75, 1.723, 7.8) \approx 1.0191923251$$
> $$\Delta _f \approx 0.0261637073$$
> $$\Delta f(x_1, x_2, x_3) \leqslant \Delta _f$$
